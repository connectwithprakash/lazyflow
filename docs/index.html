Taskweave

A calendar-first todo app for iOS that makes task scheduling as natural as writing a todo.

Features

  • Task management with lists, due dates, and reminders
  • Apple Calendar integration with drag-to-calendar time blocking
  • AI-powered task prioritization (bring your own API key)
  • Smart rescheduling when meetings conflict
  • Siri shortcuts, Apple Watch, and widgets
